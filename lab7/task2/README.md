# OOP Python Project

This project demonstrates Object-Oriented Programming (OOP) concepts in Python.

## Files
- models.py - contains classes
- main.py - runs the program

## Classes

Vehicle (base class):
- attributes: brand, speed, year
- methods: move(), info(), __str__()

Car (child class):
- inherits from Vehicle
- extra attribute: doors
- overrides move()

Bike (child class):
- inherits from Vehicle
- extra attribute: bike_type
- overrides move()

## OOP Concepts
- Inheritance
- Polymorphism
- Method overriding

## Run
python main.py
